How to use Nmap


In this article I wont go over all the Nmap commands.


What is Nmap?

Nmap (Network Mapper) is an open-source tool used for network exploration and security auditing. It is widely used by network administrators, cybersecurity professionals, and hackers alike to discover hosts and services on a computer network, thus mapping the network.


What Does Nmap do?

  • Host Discovery: Identifying devices and hosts that are up and running.
  • Port Scanning: Checking which ports are open and which services are running.
  • Service Detection: Identifying the services running on the open ports, including their versions.
  • Operating System Detection: Nmap can guess the operating system of a target device.
  • Scriptable Interaction: With Nmap Scripting Engine (NSE), users can automate tasks for vulnerability scanning and exploitation.

    • How to install Nmap

    Debian/Ubuntu:

    sudo apt install nmap


    Fedora/RHEL:

    sudo dnf install nmap

    Basic Commands

    1. Scanning a Single IP Address: To start, you can scan a single target IP address to see what services are running.

    nmap 192.168.0.1

    2. Scanning a Range of IPs:

    nmap 192.168.1.1-10
